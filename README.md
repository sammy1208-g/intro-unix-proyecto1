## Objectives
- Explain the characteristics of the UNIX/Linux operating system and compare it with other operating systems.
- Understand the file system structure in UNIX/Linux, as well as file attributes and permissions.
- Use GNU Privacy Guard (GPG) to encrypt, decrypt, and sign files and emails.
- Demonstrate proficiency in using basic UNIX/Linux commands through the terminal.
- Understand basic security principles, including disk encryption, password management, and the use of hash functions.
- Manage repositories using Git and collaborate on platforms such as GitHub and Codeberg.
